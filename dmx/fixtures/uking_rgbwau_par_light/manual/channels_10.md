# 10 Channels Mode (A001)
# Confirmed by Sina against the manual.

| Channel | Value   | Function                                                      |
|---------|---------|---------------------------------------------------------------|
| CH1     | 000-255 | Total dimming — R, G, B, W, Y, P total dimming, from dark to bright |
| CH2     | 000-255 | R dimming — Red dimming, from dark to bright                  |
| CH3     | 000-255 | G dimming — Green dimming, from dark to bright                |
| CH4     | 000-255 | B dimming — Blue dimming, from dark to bright                 |
| CH5     | 000-255 | W dimming — White dimming, from dark to bright                |
| CH6     | 000-255 | Y dimming — Amber tone light, from dark to bright             |
| CH7     | 000-255 | P dimming — Purple dimming, from dark to bright               |
| CH8     | 000-255 | Total strobe — R, G, B, W, Y, P total strobe, from slow to fast |
| CH9     | 000-050 | Function selection — CH1~CH7 control                          |
| CH9     | 051-100 | Function selection — Color output (8 options), selected by CH10 |
| CH9     | 101-150 | Function selection — Colors Jump Change                       |
| CH9     | 151-200 | Function selection — Colors Gradate                           |
| CH9     | 201-250 | Function selection — Colors Pulse Change                      |
| CH9     | 251-255 | Function selection — Sound-Active                             |
| CH10    | 000-255 | Functional speed — Function speed, from slow to fast          |
